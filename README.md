## ✨ Características

### 🛒 Item Shop

- Tienda diaria actualizada en tiempo real
- Navegación por secciones (Featured, Daily, Tracks, Cars)
- Vista detallada de ítems con precios y raridades
- Modales informativos con todos los detalles
- Contador en tiempo real hasta rotación de objetos

### 📰 Fortnite News

- Últimas noticias y actualizaciones de Epic Games
- Tabs organizadas: Battle Royale y Save The World
- Imágenes de alta calidad
- Modales con información completa

### 🗺️ Battle Royale Map

- Mapa interactivo del Battle Royale
- POIs (Points of Interest) clicables
- Modal expandible con zoom y drag

### 👕 Cosmetics

- Catálogo completo de 4000+ cosméticos
- Búsqueda en tiempo real por nombre
- Filtros por tipo, rareza, set y más
- Vista de vehículos y skins

### 🎮 Playlists

- Modos de juego históricos y actuales
- Filtros por tipo de juego
- Información de modos disponibles

### 🏴 Banners

- Galería completa de banners
- Búsqueda y filtros por categoría
- Vista previa en alta calidad

### 🎨 Diseño Moderno

- Interfaz inspirada en Fortnite
- Animaciones suaves y transiciones
- 100% responsive (Mobile, Tablet, Desktop)

## 🚀 Tecnologías

| Tecnología      | Versión |
| --------------- | ------- |
| **Angular**     | 18+   |
| **TypeScript**  | 5.5+    |
| **Bootstrap**   | 5.3   |
| **RxJS**        | 7.8+    |
| **SweetAlert2** | 11.26+  |

### Stack Técnico

- **Framework:** Angular 18 (Standalone Components)
- **UI:** Bootstrap 5 + SCSS
- **Lenguaje:** TypeScript
- **Estado:** Signals API (Angular)
- **HTTP Client:** HttpClient + RxJS
- **API:** [Fortnite-API.com](https://fortnite-api.com)
- **Deploy:** Vercel
- **Routing:** Angular Router con navegación fluida

## 🛠️ Instalación Local

### Requisitos Previos

- Node.js 18+
- npm 10+
- Angular CLI 20+

### Pasos

1. **Clonar el repositorio**

```bash
git clone https://github.com/baez-nicolas/ZeroPoint.git
cd ZeroPoint
```

2. **Instalar dependencias**

```bash
npm install
```

3. **Ejecutar en desarrollo**

```bash
ng serve
# o
npm start
```

4. **Abrir en el navegador**

```
http://localhost:4200
```

## 🎨 Características Técnicas

### Arquitectura

- **Standalone Components:** Arquitectura moderna sin NgModules
- **Signals:** Estado reactivo con Signals API de Angular
- **Services:** Inyección de dependencias centralizada
- **HTTP Client:** Llamadas asíncronas a Fortnite API
- **Type Safety:** Interfaces TypeScript para todos los modelos
- **SCSS Modular:** Estilos por componente con media queries

### API Integration

- **Recursos:** Shop, Cosmetics, News, Map, Playlists, Banners, Vehicles
- **Filtros:** Query params para búsquedas avanzadas
- **Error Handling:** Manejo robusto de errores HTTP
- **Loading States:** Indicadores de carga en todas las vistas

### Responsive Design

- **Mobile First:** Optimizado para dispositivos móviles (320px+)
- **Tablet:** Breakpoints específicos (768px-1024px)
- **Desktop:** Optimización para 1280x800 y mayores
- **Grid System:** Bootstrap grid + CSS Grid
- **Navbar:** Adaptativa con logo compacto en 1280x800

### Optimizaciones

- **Lazy Loading:** Componentes cargados bajo demanda
- **OnPush Change Detection:** Mejor rendimiento
- **Image Optimization:** Lazy loading de imágenes
- **Bundle Size:** Optimizado con tree-shaking
- **Animations:** CSS animations para mejor rendimiento
